# Changelog

All notable changes to ScreenLink are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

### Changed

- ✨ Improved project description and messaging

- 🧹 Removed all external tool references
- 🐛 Cleaned up messaging for clarity

## [1.0.0] - 2026-04-01

### Added
- ✅ Virtual Extended Monitor core functionality
- ✅ UUID-based instant link sharing
- ✅ H.264 video streaming with adaptive bitrate
- ✅ Cross-platform support (Windows, Linux, macOS)
- ✅ WebSocket real-time communication
- ✅ Tauri desktop application framework
- ✅ React frontend with TypeScript
- ✅ Node.js backend with Express
- ✅ Architecture documentation
- ✅ Protocol specification
- ✅ Development setup guide

### Known Limitations
- Single display extension per connection
- LAN-only (WAN coming in v1.1)
- No keyboard/mouse input sync (v1.2)
- No audio streaming (planned)

### How to Read This Changelog

- **Added**: New features
- **Changed**: Changes in existing functionality
- **Deprecated**: Soon-to-be removed features
- **Removed**: Removed features
- **Fixed**: Bug fixes
- **Security**: Security updates

### Version Numbering

ScreenLink follows [Semantic Versioning](https://semver.org/):
- **MAJOR**: Breaking changes
- **MINOR**: New features (backward compatible)
- **PATCH**: Bug fixes (backward compatible)

Example: `v1.2.3`
- `1` = Major version
- `2` = Minor version (new features)
- `3` = Patch version (bug fixes)
